# 2026-02-10 - Lotte Prompt Website Update Workflow

## How to Update the Lotte Prompt Website

When the user wants to add a new prompt to the website, follow this workflow:

### 1. User Provides Input
**User gives either:**
- A Twitter/X link (e.g., `https://x.com/i/status/2020961652758515772`)
- OR pastes the full prompt text directly

### 2. Extract the Prompt
**If link provided:**
- Try `bird tweet <id>` to extract
- If auth fails, ask user to paste prompt text directly

**If text provided:**
- Use the pasted text directly

### 3. Clean the Prompt
**Remove character-specific details:**
- ❌ `character_lock` (age, ethnicity, hair, eyes, body)
- ❌ `subject.description` (hair, skin, face)
- ❌ `anatomical_fidelity` (entire section)
- ❌ `body_and_anatomy` (physique, body details)
- ❌ `head_position` (character-specific)
- ❌ `features` (hair, face)
- ❌ `skin_details`

**Keep:**
- ✅ `scene` (location, environment)
- ✅ `subject.pose` (orientation, limbs, spine)
- ✅ `outfit` / `wardrobe` (clothing descriptions)
- ✅ `lighting` (type, quality, direction)
- ✅ `camera` (shot_type, angle, focus)
- ✅ `mood_and_expression` (emotion, mood, atmosphere)
- ✅ `expression` and `gaze` (as requested by user)

### 4. Add to Collection
**File:** `prompts_data.json`

**Determine category:**
- `Explicit` - thong, bikini, lingerie, suggestive
- `Lingerie & Intimate` - less explicit lingerie
- `Bedroom & Morning` - bed, bedroom scenes
- `Bathroom & Shower` - shower, tub, bath
- `Outfits & Costumes` - dresses, costumes
- `Other` - everything else

**Add as:**
```json
{
  "id": "manual_X",
  "text": "Caption text\n\nNano Banana Pro\n\n{json}",
  "source": "manual"
}
```

### 5. Commit and Push
```bash
cd /home/gavien/workspace/lotte
git add prompts_data.json
git commit -m "Add [description] prompt - total XXX"
git push promptlib main
```

### 6. Website Updates
- GitHub Pages auto-deploys in 1-2 minutes
- User should hard refresh (Ctrl+F5) to see changes
- Website: https://adamfiverr-lab.github.io/lotte-prompt-library/
